# LLM Chess app


## Summary

This is a vibe-coded toy project. The objective is to create scaffolding so that LLMs can play chess correctly.

The main idea is a cli tool that can hold a game in memory and know allowed moves,
perform moves and print the board state. 


## Languages and Tools 

- All python, will be a cli tool.
- use uv, ruff, ty
- We will use python-chess library
